***************** Installation *****************

Schritt 1. 
Öffne die Datei config.demo.inc.php mit einem
Texteditor und ändere alle Einstellungen 
entsprechend der Vorgaben. Danach speichere die 
Datei als config.inc.php im gleichen Verzeichnis.

Schritt 2.
Lade allen Dateien mit einem FTP-Programm in 
das Wurzelverzeichnis(root) auf deinen Server.

Schritt 3.
Rufe mit einen Browser deine Domain auf. Um
die Installation zu starten, muß das zuvor in
der Konfigurationsdatei vergebene Passwort
eigegeben werden.

Schritt 4.
Nach erfolgreicher Installation logge dich
in den Administratorbereich ein und lege 
einen neuen Admin an. 
WICHTIG! Danach sollte der User 'webmaster'
unbedingt gelöscht werden.

************* System Anforderungen *************

PHP Version 5.5.0 oder höher.
MySQL Version 5.0 oder höher.

Hinweis: Das kannst das mod_rewrite Apache Modul 
für Suchmaschinenfreundliche URL nutzen.
